Clon Mac Mini M1
La Landing Page del Mac Mini M1 es un sitio web inspirado en la computadora de Apple. Para crearlo, utilicé las tecnologías HTML, CSS y JavaScript. A lo largo del proceso de desarrollo, aprendí a implementar animaciones y a utilizar las Dev Tools del navegador para depurar y optimizar el sitio.
Definí los elementos como encabezados, párrafos, imágenes y enlaces utilizando HTML.
Diseñé el aspecto visual de la Landing Page utilizando CSS. Definí estilos para los elementos HTML, como colores, fuentes, márgenes y tamaños. Aprendí a aplicar reglas de diseño responsivo para que la página se vea bien en diferentes dispositivos y aplicar variables.
Implementé interactividad y dinamismo en la página mediante JavaScript. Aprendí a manipular el DOM. Agregué eventos y crear animaciones.
